BitSatCredit is a Lightning-powered credit system for the BitSatRelay satellite messaging network.

Users can top up their accounts via Lightning payments and use credits to send messages through the satellite network. All transactions are tracked and balances are managed automatically.

Features:
  • Lightning-powered top-ups with instant crediting
  • User balance and transaction tracking
  • REST API for BitSatRelay integration
  • Public top-up page for self-service
  • Admin dashboard for monitoring
Part of the BitSatRelay project by CryptoIceMLH.
